Descrição detalhada

Esta classe é responsável pela comunicação entre o HUB e o banco de dados firebase.

Este módulo provem funções básicas de comunição, entretando não é responsável pela lógica usada de acordo com as mensagens.

Parâmetros
hubIDO id do hub cadastrado no banco de dados firebase
appIDO id do app/do usuário do app no banco de dados firebase que possui o hub.
firebaseObjeto da classe pyrebase, responsável pela conexão com o servidor
databaseObjeto da classe pyrebase, responsável por recuperar dados do banco de dados firebase.
mensagensRecebidasAs mensagens recebidas pelo módulo, guardadas como uma lista ordenada.
