Descrição detalhada

Classe para o LedManager.

O LedManager tem seus métodos chamados pelo método acionarLed do HUB. Esta classe deve servir de apoio para a classe correspondente no HUB. Não há um loop principal.

Parâmetros
coresDicionário que contém os status e cores, onde para cada status há uma cor correspondente
Aviso
Adicionar os atributos dessa classe correspondentes a classes necessárias para de fato acionar o led em hardware, e inicializar os mesmos em init.

Método de inicialização para o LedManager.
