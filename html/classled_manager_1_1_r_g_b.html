Descrição detalhada

Classe para rgb.

Essa classe nada mais é do que o equivalente a uma struct em c. Só armazena de forma rezumida uma cor RGB.

Parâmetros
redCor vermelha armazenada, inteiro, com valores indo de 0 a 255
greenCor verde armazenada, inteiro, com valores indo de 0 a 255
blueCor azul armazenada, inteiro, com valores indo de 0 a 255
